📖 Instalación

🐧 Linux — AppImage

Requerimientos: WebKitGTK 4.1 (lo trae casi cualquier distro moderna).

  1. Descargá el archivo .AppImage.
  2. Hacelo ejecutable: chmod +x ClassDeck*.AppImage
  3. Ejecutalo con doble clic o desde la terminal.

💡 También podés moverlo a ~/Applications y crear un acceso directo.

🐧 Linux — Debian/Ubuntu (.deb) 
sudo dpkg -i ClassDeck*.deb

Si faltan dependencias: sudo apt install -f

🐧 Linux — Fedora/RHEL (.rpm) 
sudo rpm -i ClassDeck*.rpm

O con dnf: sudo dnf install ./ClassDeck*.rpm

🪟 Windows

Ejecutá el .exe y seguí las instrucciones del instalador.

También está disponible el .msi para instalación silenciosa o en red.
